# @papit/foundations-group

Behavioral grouping primitives for managing focus and keyboard navigation across related DOM elements, independent of visual layout or rendering framework.

## installation

```bash
npm install @papit/foundations-group
```

### to use in **html**

```html
<script type="module" defer>
  import "@papit/foundations-group";
</script>

<pap-group></pap-group>
```
